# css-utilities
A collection of simple css utility classes

Build it with: `npm run build`

```
css
├─── utilities.css
└─── utilities.min.css

scss
├─── utilities.scss
│
└─── utilities
        ├─── _display.scss
        ├─── _flexbox.scss
        ├─── _float.scss
        ├─── _image.scss
        ├─── _overflow.scss
        ├─── _position.scss
        ├─── _shadows.scss
        ├─── _sizing.scss
        ├─── _spacing.scss
        └─── _text.scss
```
